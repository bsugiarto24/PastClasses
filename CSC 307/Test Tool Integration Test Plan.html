Test Tool Integration Test Plan


After the classes in each package are independently tested together, perform integration testing in the following phases:

  1. Integrate the TestToolDB with the AdministrativeDetails package.

  2. Integrate the TestToolDB with the QuestionCreation package.

  3. Integrate the TestToolDB with the QuestionBank package.

  4. Integrate the TestToolDB with the TestCreation package.

  5. Integrate the TestToolDB with the TestBank package.

  6. Integrate the TestToolDB with the TestTaking package.

  7. Integrate the TestToolDB with the TestGrading package.

  8. Integrate the packages and make sure they work together when added together in the order above.

  9. Add the top-level TestTool classes to the integrated packages and run the top-level class tests, which in turn run the class tests in each package.
