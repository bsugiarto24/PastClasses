1.5.1. ClassMarker

ClassMarker is an online quiz maker used by both bussiness and schools. It is very intuitive and easy to use as quiz creation can be done quickly. ClassMarker makes it easy to customize quizes to specific clientele by allowing uses to import images, videos, and external webpages to questions. It also has api's that can be added so that tests can be taking on different websites.

ClassMarker seems to be an industy leader because of its easy of use. The test creation workflow is very intuitive and can be used by those that may not be very tech savvy.

Good Features:

  • Allows importing of images and videos for each question
  • Makes test creation very easy and straightforward.
  • Has test settings page where a test creater can choose settings such as:
    • Tries Allowed
    • Time Limit
    • Randomize order of questions and answers
    • What's included on the results page
    • What is the score cutoff for a passing score
  • Automates grading and allows immidiate feedback
  • Grades can be easily exported to an excel spreadsheet
  • Tests can be integrated into customers websites through the use of apis
  • Allows tailored messages for each question which can either say what the test taker did right if he/she got the quesiton correct or more details as to why they got the question wrong.

Bad Features:

  • A subscription based service there is a free tier, but question creation and the number of tests given per month are limited.

Missing Features:

  • Does not allow the user to save questions to form a test bank
  • Does not allow a code-based question type
  • There are not time limits on specific questions just a time limit for the whole test
