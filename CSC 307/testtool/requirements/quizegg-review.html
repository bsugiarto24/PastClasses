1.5.2. QuizEgg

QuizEgg is a web-based application made for easy test creation and test taking. It offers a wide range of question types, as well as a the ability to include files such as Word documents and pictures to accompany questions. QuizEgg also includes a variety of quiz setting features such as the question randomization, when the answers to a question are revealed, and limiting which users can take a quiz.

Because QuizEgg is all web based, it provides test makers the ability to easily create diverse and challenging quizzes, and test takers a simple way of taking quizzes.

Good Features:

  • Supports 8 question types:
    • Multiple Choice
    • True/False
    • Multiple Response
    • Fill in the Blank
    • Word List
    • Matching
    • Ordering
    • Essay
  • Offers a variety of quiz settings:
    • Question Randomization
    • Number of Retakes
    • Timer
    • Showing Answers After Each Question
    • Showing Answers at End of Quiz
    • Limit which users can take quiz
  • Allows files to be uploaded to accompany questions
  • Customizable themes for quiz web-page
  • Organized report of test takers and their respective test results that can be exported to an Excel file
  • All web-based

Bad Features:

  • Testing can only be done through their website, unless you opt to print out the test
  • Not free, $50 yearly subscription fee
  • Themes are very simple and limited to background color, font color, and a logo

Missing Features:

  • Question type: Choose All That Apply
