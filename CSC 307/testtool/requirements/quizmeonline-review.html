1.5.5. QuizMeOnline

QuizMeOnline is a free online tool that allows users to create tests and develop study guides to study for exams. The tool allows the user to create, share, and grade quizzes. The objective of the website is meant to help students study together instead of creating tests for school. There are several features missing such as test reports and certain question types.

Overall, QuizMeOnline has a look and feel of a cheap free online tool that should not be relied on for school use to administer tests.

Good Features:

  • Free
  • All the basic test tool functionality exists
  • Ability to create study groups from study buddies
  • Can study from study guides and flashcards
  • Tests automatically grade

Bad Features:

  • Games on the website which may distract user from studying
  • Commercials on the website

Missing Features:

  • Test questions are not categorized
  • No test generation feature
  • No test report after
  • Question type: Choose All That Apply
