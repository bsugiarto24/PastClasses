2.4. Question Settings

When the user initially invokes the Test Generation, the screen appears as shown in Figure 1.

Figure 1: Initial screen.


The initial display will consists a list of variables a teacher can be used to generate a test. It will contain at the minmum: For Class (which class this test is for), Test Name, Difficulty, Question Types, and much more.

Figure 2 shows how to edit.

Figure 2: Expanded Menus Tab.


Click on the drop down menu of a setting. Choose a desired level for that specific setting.

At the end, click "Create" to create the test.
