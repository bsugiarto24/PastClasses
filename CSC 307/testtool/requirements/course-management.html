2.8.2 Admininstrative Actions For Students


This view is shown when a student is logged in. To view details of a course, the student should click on the More Details button in the row. This way, the student will be linked to the statistics portion of the application which will show their test scores. To delete a course, the student will click on the Minus button.

Figure 7: Viewing Students' Courses


To add a course, the user should select the Add Course button in the bottom row. By clicking the row, a form will pop up to take in a Course ID or Entry Code. To add a course, the user must input a proper Course ID. If an incorrect Id is entered, a pop up shown in Figure 9 will be displayed.
