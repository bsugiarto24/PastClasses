2.4.1. Adding, Editing, Deleting a Question in Question Bank

To add a question, the user will click on the 'Add' button. This will take them to the add a question screen where they will follow the instructions to create a question.

Figure 3-12

To edit a question, the user will click on the 'Edit' button as shown in the figure below.

Figure 3-13

This will pull up a text boxes for the user so the user can make any changes to the question as shown in the figure below.

Figure 3-14

To delete a question the user will start with this screen that have questions in the question bank.

Figure 3-15

The user will then press the 'Delete' button for the question that they want to delete which will pull up an Alert Box prompting the user if he or she wants to delete the question.

Figure 3-16

If they user selects 'Yes', the question will be removed as such.
