1.5.4. Hot Potatoes

Hot Potatoes is a Java application that allows for test generation with user defined questions. Because it is a Java application, it can be used on any machine that has a Java Virtual Machine, making it multi-platform. It supports integration with many types of tests and the combination of different test types in one html document.

Overall, Hot Potatoes provides a standard level of testing functionality. It does not support automated test generation, since there is no question bank.

Good Features:

  • Question types:
    • Multiple choice
    • Gapfill with check and hint
    • Essay free response questions
    • Gapfill with dropdown and individual questions
    • Crossword
    • Mixed-up sentence (supports undo one move and restart)
    • Mixed-up word
    • Matching with drop-downs
    • Drag-and-drop matching
  • Provides a Help tab with tutorials
  • Prompts you to save file upon hitting exit (not losing content)
  • Prompts 'hints' for questions when unable to answer question
  • Ability to re-take test (if professor supports) with persisting answers
  • Ability to embed Links and Images
  • Interface for creating 5 specific 'Jtypes' of tests, and then a 'Masher' to combine all the questions into a test.
  • 'Masher' allows for customization of test (size, fonts, colors, etc.)
  • Paths for different projects to save all tests in one directory

Bad Features:

  • Undescriptive features - you have to click around to learn how everything works
  • Buggy UI when resizing window (doesn't resize when maximizing then minimizing)
  • A little overcomplicated - would definitely take some time to learn
  • Undescriptive fields (main vs alternate sentence, etc.)

Missing Features:

  • Reports to display to the professor and students
  • Question bank for test generation
  • Ranking questions (difficulty, length, etc.)
  • Automated test generation based on question bank
